# NFT > NFT全称为non-fungible Token，中文名叫非同质化通证。我们常见的Token（如BTC，ETH等）都是同质化的，每个BTC之间没有任何区别（就像2张100元人民币一样），可以互换和分割。而NFT的重要特征在于：每一个NFT拥有独特且唯一的标识，两两不可互换，最小单位是1且不可分割。

NFT是一种数据协议，它由链承载，链上流通，加密确权。它所应用到的业务也并不一定需要是去中心化的。

NFT的作用

货币本质上是一般等价物，是具有价值共识的载体.对于现代社会的一般个体来说，资产所承载的价值旺旺大于其持有的现金价值。

很多实体资产本身是一种中心化资产，在NFT未获取共识之前，依赖中心化机构的背书

NFT的使用

  • 加密数字经济是由NFT和FT共同构成的——FT是流动的血液，NFT是坚韧的基石。
  • 正确地使用NFT
    • 在数据底层，NFT应是immutable的——一旦发行不可更改，始终确权，保证了资产本身的安全性；
    • 在应用层，主要制定对NFT的描述逻辑，即如何正确识别NFT
    • 识别逻辑可以是中心化的，构成了同一个NFT的不同应用场景，正所谓“资产复用”

现实世界资产的缺陷

  • 资产流动性差
  • 方位成本高。部分资产需要中心化确认

NFT替代方案

在区块链世界中，加密货币对应现实中的货币，而NFT对应现实中的非货币资产。

  • 流动性更好，高效、原子化交易
  • 防篡改
  • 额外优势：表征负债，身份和权益证明，虚拟道具物品

NFT 使用场景

NFT可以表征数字资产本身，也可以表征某种特殊用途的凭证（门票、房产证、护照等）

游戏道具

游戏道具是非常适合采用NFT来进行表征的数据形式。

  • 游戏道具产自链上，用之链上，由游戏场景提供价值支撑，规避了“价值上链”这一中心化的环节
  • 大部分的游戏装备、角色都是独一无二的，且能够自由流通，这与NFT的特性相符

收藏品

收藏品有两个特性：独一无二与不可伪造,在艺术品领域，我们可以将对书画、雕塑等艺术资产的所有权以NTF的形式保存在区块链上，将实体寄存于一些强信任的中心机构（如博物馆、金库等），便可实现在不需要接触实物的情况下进行确权、交易、认证、流转等操作.甚至能将艺术品证券化，成为一种流动性更高的投资品.

主流标准

ID(全局id)，NAME(名称)，SYMBOL(符号)，URI(统一资源标识符)。其中SYMBOL和Name是可选的

NFT 讨论

绘画艺术品， 艺术品本身的数字文件存在哪里？

  • 业内许多是存在ipfs上的，如果数据比较小，比如一些构成艺术品本身可以用某些算法或者参数定义，可以直接存链上合约

如果不在链上 没法防止复制？

  • 也不是防止复制，你复制了也只是一份copy，就比如你幅名画，你临摹的再像，也是赝品
  • NFT不一样，比如梵高画了一个画，用自己的签名发在链上。以后只会在链上转移此画的归属权，而且可追溯，不管交易多少手，别人都知道是梵高画的画

如果不在链上 实际上 没法确定真伪？链下我复制一个数据文件 数据 是否是真品 没发确定？

  • 就是你链下拿到这个数据没用，实际链上这个数据也不属于你。你只是类似给画拍了张照片而已

这个数据是一份程序镜像 我复制了 就能用了？

  • 我用之前去链上查一下这个镜像hash的拥有者是谁，如果是你我就让你用
  • 如果你不是在我这用，你要在你自己机器上用，那我没办法。就像你把一副假画挂家里一样
