# 《区块链：定义未来金融与经济新格局》 作者火币网 张健

https://weread.qq.com/web/reader/bb5322005cad2dbb59d0bf8kc9f326d018c9f0f895fb5e4

区块链产生背景

区块链使用技术的方式为交易双方建立信用，而无需第三方机构参与，从而极大地降低了交易成本

计算机技术发展，使得数字化货币流通成为了可能，区块链构建了数字化货币的去中心化流通模型。

数字货币

巴别塔：

起初人类有共同的语言，并且一起居住在与幼发拉底河相距不远的地方。人们利用河谷的资源在那里建筑城和塔，以聚集全体人类。上帝降临视察，认为人类过于自信和团结，一旦完成计划，人类将无所不能。上帝决定打乱人们的口音和语言，并使他们分散各地。于是高塔停工了，人们操持不同的语言，互相之间难以交流。这个塔就是巴别塔。

数字货币：货币的巴别塔？

信用与货币

金钱正是有史以来最普遍也最有效的互信系统

信用是货币的制造者，人们相信货币具有购买力的故事（本质是价值交换）。不管是实体货币还是虚拟货币，人们需要的不是货币本身，而是货币本身建立的信任购买力体系。

信用的计算？

潜在收益×信用行为的概率 > / < / = 潜在损失×不守信行为的概率

区块链不计算人或者参与主体的信用，而是计算信用行为（交易）的可信程度。区块链可以方便计算违约成本和违约收益（例如挖矿），帮助构建信用体系。

互联网数字货币去中心化

互联网带来了信息交换的能力，使得数字货币化有了可能。然后数字货币化传统上依附于第三方可信机构（银行/淘宝），区块链提供了去中心化的数字货币流通能力。

比特币的诞生背景

比特币诞生离不开一个密码朋克（CipherPunk）的组织，这个组织旨在倡导去中心化、匿名、保护隐私等

比特币从组织中借鉴的思想：

  • POW 工作量证明（Proof Of Work），要求计算机在做某件事之前需要进行一定的计算工作。

  • 时间戳 时间戳表达文件的创建先后顺序，文件创建后，时间戳不能改动。

  • P2P 每个点都维护整个交易，节点之间是平等的

区块链是什么

未来，区块链会营造价值互联网。（卖家和买家直接交易）。

记账货币

区块链货币（比特币）基于记账货币

当在网银转账的时候，没有发生货币的转移，只是银行的账本发生了变化。

货币本身没有价值，他所承载的信用体系中的购买力才具有价值。

去中心化记账方式

1）账本数据的存储必须是去中心化的，不能指定任何参与方拥有特殊的保存账本的权力，或者说，我们需要让所有参与方都平等地拥有保存账本的权力。

2）记账行为本身必须是去中心化的，不能指定任何参与方拥有特殊的记账权力，或者说，我们需要让所有参与方都平等地拥有记录账务数据的权力。

比特币通过POW（也成为挖矿）的方式，实现了去中心化的记账：

就是以每个节点的计算能力（“算力”）来竞争记账权的一种机制。在比特币系统中，大约每十分钟进行一轮算力竞赛(POW)，竞赛的胜利者获得一次记账的权力，即向区块链这个总账本写入一个新区块的权力,写入新区块完成记账的节点会获得奖励（比特币的发行过程）

共识机制与价值载体

传统的系统结构的共识，由中心决定，较为高效。区块链则使用共识机制

对于公链，共识机制需要消耗一定的资源（例如比特币需要POW）

区块链不仅仅能用于数字货币，也可用于金融、债券等可以以二进制表示的资产的流通。

智能合约

区块链上的交易，是由定义的智能合约实现的，因此可以实现一些额外的交易逻辑。例如多方签名。

多方签名是指一个资产对应多份私钥，必须签名的人达到了一定的比例才能动用该资产

侧链

侧链是不同链之间的通路，可以实现链之间的资产流动

数字货币产业链

上游：挖矿
矿机、矿池、云挖矿

中游 ：交易平台

下游： 比特币钱包 比特币支付

互联网金融

互联网金融并不是对传统金融概念的革新，二者理念并无区别。区别在于，互联网金融通过技术的方式，提供了消灭信息不对称的金融途径。

区块链技术在此之上，提供了去中心化、降低金融成本、提升效率、降低风险的金融途径。

目前互联网金融的基础设施尚不完善，但是前景比较开阔。

物联网和共享经济

物联网：通过信息传感设备，按照约定的协议，把任何物品与互联网连接起来，进行信息交换和通信，以实现智能化识别、定位、跟踪、监控和管理的一种网络。在物联网时代，能接入网络的不仅仅是计算机和智能手机，还有汽车、健身设备、锁具、交通摄像头甚至你能想到的一切

物联网技术促进了共享经济的兴起，共享经济需要建立可信、隐私的网络结构，区块链可以提供帮助。

区块链基础设施

在社会和经济生活领域，基础设施是指为社会生产和居民生活提供核心公共服务的系统，这些基础设施往往具有先行性和基础性，若没有它们，其他生产和生活活动就难以开展。基础设施通常只有达到一定规模时才能提供有效的服务。

当今区块链行业处于早期阶段，最终的是构建起区块链产业的基础设施。构建的具体内容是1、多种类型、适用多种场景的链 2、侧链技术。核心难点是如何解决统一标准的问题。

区块链的具体使用场景

数字货币

各国态度：大致认为比特币是一种“货物”，允许个人进行流通，但禁止银行或是企业流通。

众筹、版权、公证与记录

众筹主要是透明、便捷筹资

版权突破地域限制，方便维护版权所有者权益

公正与记录提供了一种方便的证明手段（区块链的不可篡改特性）。例如把你的毕业证、驾照放到区块链上。

清算、结算与审计

金融领域：结算是指各经济单位由于商品交易、劳务供应和资金调拨等经济活动而引起的货币收付行为。按支付方式不同可以分为3种：现金结算、票据转让和转账结算。在当前的实践中，结算更接近不动产清算中的产权转移过程或者一个交易被执行后金融机构交换对价[插图]的过程[插图]。而清算则多是涉及银行间的资金结算，一般为联行业务。审计是指由专设机关依照法律对国家各级政府及金融机构、企业/事业组织的重大项目和财务收支进行事前、事后审查的独立性经济监督活动

区块链带来的好处：

  • 增加透明度、降低信任成本
  • 过程自动化、缺少中间环节
  • 分布式账本，防御单点故障
  • 满足监管和审计要求

智能合约

合同（Contract）又称契约、协议，是平等主体的自然人、法人、其他组织之间设立、变更、终止民事权利义务关系的协议[插图]。本质上讲，合同是当事双方或多方在并没有充足信任的情况下，通过文字的约定和法律的权威，对各自的权利与义务进行的约定。

根据区块链可编程的特点，人们可以将合同变成代码的形式放到区块链上，并在约定的条件下自动执行，这就是所谓的智能合约

智能合约面临的问题

  • 基于区块链的数字化资产不过。
  • 智能合约在去中心化的系统中自动执行，不仅要求智能合约的载体去中心化，执行合同的出发条件也应当是去中心化的。

密码学

区块链构建

地址：目的是简化标定一个主体需要的长度。一个地址相关的交易，需要用这个地址对应的私钥签名

在比特币体系里，椭圆曲线签名算法里的私钥由32字节随机数组成，通过私钥可以算出公钥，公钥经过一系列哈希算法及编码算法就得到了比特币中的地址。因此地址其实是公钥的另一种表现形式，可以理解为公钥的摘要。
